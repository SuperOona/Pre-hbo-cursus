## Getting Started

Steps on how to use this starter template

### Installation

1. Installing 
   ```
   pip install -r requirements.txt
   ```
2. Write you're bot credentials
   ```json
   {
    "token": "token",
    "prefix": "prefix"
    }
   ```
4. Run you're bot.
   ```
   python main.py
   ```
